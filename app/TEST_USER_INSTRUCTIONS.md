# Инструкция по использованию тестового пользователя

## Быстрый старт с QR-кодом и TOTP

### 1. Создание тестового пользователя
1. Откройте приложение
2. На экране входа нажмите кнопку "Создать тестового пользователя"
3. Дождитесь создания пользователя - появится QR-код

### 2. Настройка приложения аутентификации
1. Скачайте приложение аутентификации:
   - Google Authenticator (Android/iOS)
   - Authy (Android/iOS)
   - Microsoft Authenticator (Android/iOS)

2. Добавьте новый аккаунт:
   - В приложении аутентификации нажмите "+" или "Добавить аккаунт"
   - Выберите "Сканировать QR-код"
   - Наведите камеру на QR-код, отображенный в приложении

### 3. Вход в приложение
Используйте следующие данные для входа:

**Логин:** `testuser`  
**Пароль:** `testpassword123`  
**TOTP код:** 6-значный код из приложения аутентификации

### Способы входа:
1. **Стандартный:** Введите логин, пароль и TOTP код
2. **Только TOTP:** Введите логин и TOTP код (без пароля)

### Проверка работы:
1. После успешного входа вы должны увидеть главный экран приложения
2. Если появляется ошибка "Неверные данные аутентификации", проверьте:
   - Правильность ввода логина и пароля
   - Синхронизацию времени на устройстве
   - Правильность TOTP кода (6 цифр)

### Если что-то не работает:
1. Убедитесь, что время на вашем устройстве синхронизировано
2. Попробуйте сгенерировать новый QR-код
3. Проверьте, что приложение аутентификации настроено правильно

### Технические детали:
- TOTP секрет генерируется автоматически
- QR-код содержит ссылку в формате otpauth://totp/testuser?secret=...
- Коды действительны 30 секунд
- Поддерживается временное окно ±30 секунд для синхронизации
